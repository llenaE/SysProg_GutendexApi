# SysProg_GutendexApi
Treći projekat - Sistemsko programiranje
## Zadatak 28:
Koristeći Rx.NET i Project Gutenberg (Gutendex) API, prikupiti opise knjiga (summaries) za
zadatog autora. Rx vrši osnovno mapiranje dobijenih podataka i emituje ih kao poruke aktorima.
Aktori čuvaju opise knjiga kao interno stanje i implementiraju Topic Modeling koristeći ML.NET
biblioteku. Web server prima zahteve sa imenom autora i prevodi ih u poruke aktorima. Prikazati
identifikovane teme i dobijene rezultate.
Dokumentacija dostupna na linku: https://gutendex.com/
